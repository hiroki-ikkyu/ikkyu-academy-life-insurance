EV（エンベディッドバリュー）分析

生保の「真の価値」を測る

EV = 修正純資産(ANW) + 保有契約価値(VIF)

通常の会計利益では保険契約の長期的価値が見えない。EVは将来キャッシュフローの現在価値を含めた包括的な企業価値指標。

  • 修正純資産(ANW)：純資産＋有価証券含み損益＋負債の含み損益等
  • 保有契約価値(VIF)：既存契約の将来利益PV。金利・死亡率前提で大きく変動
  • MCEV：EEV原則に準拠した市場整合的EV。フリクショナルコスト・CRNHR含む
ALM（資産負債管理）

生保経営の心臓部

保険契約（負債）のキャッシュフローと運用資産のキャッシュフローをマッチングさせる管理手法。

  • デュレーション・ギャップ：負債デュレーション（20-30年）vs 資産デュレーション（10-15年）
  • 金利リスク：金利上昇→資産の時価下落、しかし負債も減少（ネットで利益のことが多い）
  • 超長期債の役割：30年・40年国債でギャップを縮小
  • 為替ヘッジ：外債投資の為替リスクをどこまでヘッジするか
三利源（Three Profit Sources）

生保利益の三本柱

基礎利益 = 死差益 + 利差益 + 費差益
  • 死差益：予定死亡率 > 実際死亡率 → 死差益。日本では長寿化で構造的にプラス
  • 利差益（損）：運用利回り vs 予定利率。低金利時代は逆ざや（利差損）が問題だった
  • 費差益：予定事業費率 > 実際の事業費率 → 費差益。効率化で改善傾向

KEY 基礎利益は生保版の「営業利益」。ここから臨時損益（有価証券売却損益等）を加減して経常利益へ。

ESR・ソルベンシー規制

2025年度〜新規制スタート

ESR = 適格資本(Own Funds) ÷ 所要資本(SCR)

資産・負債をすべて経済価値（時価）で評価し、ストレスシナリオ下でも支払能力があるかを検証。

  • 従来のSMR：簿価ベース。200%以上で「健全」
  • 新ESR：時価ベース。100%以上が必要、150-200%が一般的なターゲット
  • 金利感応度：金利1%上昇→負債の時価が大きく減少→ESR改善
  • 株式リスク：政策保有株の多い会社はESR毀損リスク大
新契約分析

成長性を測る鍵

新契約価値 = 新契約の将来利益PV
新契約マージン = 新契約価値 ÷ 保険料PV
  • 新契約年換算保険料(ANP)：年間ベースでの新契約規模
  • 新契約価値：その年に獲得した契約の生涯価値
  • マージン：8-12%が日系大手の典型。保障性商品は高マージン
  • 第三分野比率：医療・がん保険の比率が高いほどマージン改善
バリュエーション手法

生保株の適正価格をどう測るか

  • P/EV倍率：時価総額÷EV。0.5x以下は割安と見る向きが多い
  • APEV法：EV + 新契約価値×成長倍率。成長性を加味
  • P/EV + 新契約倍率：新契約価値に何倍のマルチプルを置くか
  • DDM：配当割引モデル。還元強化トレンドで注目度上昇
  • Sum-of-Parts：国内生保＋海外子会社を分離評価

注意 相互会社は非上場のためEV開示が主な情報源。株式会社4社（T&D, 第一生命, かんぽ, SOMPOひまわり）のみ株式市場で取引。
