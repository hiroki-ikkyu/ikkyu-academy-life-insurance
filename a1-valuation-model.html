バリュエーション・モデル構築
APEV・P/EV・新契約倍率・DDM・SoP——生保株の適正価格を算出する

📐 生保バリュエーションの基本フレームワーク

生保はPERだけでは評価できない。帳簿上の利益と経済的な企業価値の乖離が大きいため、EV/MCEVベースのバリュエーションが不可欠。

主要手法の一覧

  • P/EV倍率：最も基本。時価総額 ÷ EV
  • APEV法：EV + 新契約価値 × 成長倍率（マルチプル）
  • P/EV + VNB Multiple：分離的にEVストックと成長（VNB）を評価
  • DDM（配当割引モデル）：配当の持続性とペイアウト率で評価
  • Sum-of-Parts：国内＋海外事業を分離評価

💎 P/EV倍率の実務

P/EV = 時価総額 ÷ EV

P/EVの読み方

  • < 0.5x：市場がEVの信頼性を疑っている or 深刻な構造問題
  • 0.5-0.8x：日系生保の典型レンジ。割安に見えるが「EVディスカウント」が常態化
  • 0.8-1.0x：適正～やや割高。EVの質が高い（高マージン新契約、安定的VIF）
  • > 1.0x：強い成長期待 or 海外事業の高評価

なぜ日本生保はP/EV 1.0x未満なのか？

  • 新契約のマイナス成長懸念
  • 金利前提への不信（低金利が長期化する可能性）
  • 相互会社との競争による価格圧力
  • ESR規制導入の不確実性
  • 資本効率の低さ（ROE < CoE）

📊 APEV法の構築

理論株価 = EV + VNB × Multiple
Multiple = (1 + g) / (RoEV − g)

g: VNBの長期成長率、RoEV: EVに対する期待リターン

実務上のステップ

  1. 最新のEVを取得（開示資料から）
  2. VNBの正常化（一時的要因を除去）
  3. 長期成長率gの設定（0-3%が典型）
  4. 割引率RoEVの設定（8-12%。リスクフリー+エクイティリスクプレミアム）
  5. マルチプルを計算→理論株価を算出
  6. 感応度テーブルを作成（g vs RoEVのマトリクス）

🧮 Sum-of-Parts分析

海外事業比率が高い会社（第一生命等）には不可欠。

理論株価 = 国内EV × P/EV_国内 + 海外EV × P/EV_海外 + 持ち株/余剰資本
  • 国内事業：P/EV 0.4-0.7x（成長なし→ディスカウント）
  • 海外事業：P/EV 0.8-1.2x（成長プレミアム）
  • グループ調整：持株会社コスト、余剰資本、のれん

🧪 確認クイズ

APEV法で理論株価を計算する際、VNBの成長倍率（マルチプル）は何の関数か？

✅ APEV法のマルチプル = (1+g) / (RoEV − g)。成長率が高く期待リターンが低いほどマルチプルは高くなる。永久成長モデル（ゴードンモデル）と同じ構造。
